Welcome to Lindes CT

Using the Residual Encoder Decoder Architecture we have trained a RED-CNN model to denoise CT scans containing Poisson, Periodic and Gaussian noise. The model can denoise the image within seconds with high accuracy and throughput, which ultimately aids in diagnosis and research. We have curated datasets from various trusted medical data repositories

The encoder extracts hierarchical features from the noisy CT scan, capturing both low-level details (e.g., edges) and high-level structures (e.g., organs). The decoder reconstructs the clean image from these features, progressively refining the output to remove noise while preserving critical anatomical details. By effectively denoising low-dose CT scans, our solution allows doctors to use lower radiation doses without compromising image quality, enhancing patient safety.
